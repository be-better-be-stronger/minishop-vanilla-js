# MiniShop - Vanilla JavaScript

A simple product catalog + cart demo built with HTML/CSS/JavaScript (ES6).
Cart is persisted with LocalStorage.

## Features
- Product grid + search
- Filter by category
- Sort by price/name
- Product detail modal
- Add to cart, update quantity, remove items
- Cart drawer + total calculation
- LocalStorage persistence

## Run
Open `index.html` in a browser.
Click "Seed Demo" to generate sample products for screenshots.
